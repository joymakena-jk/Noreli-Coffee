# Noreli Coffee

Noreli Coffee is a modern café-inspired front-end website that blends warmth, elegance, and comfort into a welcoming digital experience. The project was built to strengthen my understanding of HTML, CSS, and JavaScript by creating a fully responsive and visually engaging website from scratch.

## Features

* Cinematic full-screen hero section with typewriter animation
* Responsive navigation bar with smooth scrolling
* About section showcasing the café's story and values
* Signature Coffee collection
* Fresh Bakery showcase
* Reading, Working, and Outdoor Patio spaces
* Interactive image gallery
* Guest testimonial slider with automatic transitions
* Scroll reveal animations
* Cursor glow effect
* Button ripple animations
* Back-to-top button
* Active navigation highlighting
* Responsive design for desktop, tablet, and mobile devices
* Premium café-inspired interface with elegant hover effects

## Built With

* HTML5
* CSS3
* JavaScript (Vanilla JavaScript)

## Project Goals

The goal of this project was to:

* Practice semantic HTML structure
* Improve CSS layout, responsiveness, and animations
* Strengthen JavaScript DOM manipulation and event handling
* Build a fully responsive website without using front-end frameworks such as Bootstrap
* Create a modern café-inspired user interface with interactive features and smooth user experience
